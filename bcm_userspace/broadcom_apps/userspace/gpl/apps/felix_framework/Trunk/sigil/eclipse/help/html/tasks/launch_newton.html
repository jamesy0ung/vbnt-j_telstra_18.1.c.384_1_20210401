Runtime Configuration

To run a Infiniflow Service Fabric container within Eclipse you first need to configure its runtime options in the IDE.

Open the Run configuration dialog by selecting the Run → Run… option from the menu bar.

Create a new Newton configuration by right-clicking on the Newton configuration group and selecting New. Type in a name for your configuration, the other default options should be sufficient for most users.

Launch Newton

Click the Run button to start the container, you will see the container output appear in the console tab. Wait until the Boot Complete line appears, as below.
