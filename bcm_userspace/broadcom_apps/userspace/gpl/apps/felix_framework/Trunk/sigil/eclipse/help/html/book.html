Sigil - Newton Development Plugin

Sigil is an eclipse plugin that helps developers build and test code to be deployed on the Newton platform

Features:

  • New (Newton Project, Composite, System)
  • Syntax highlighting and code completion for SCA/System docs
  • Launch Newton in Debug/Standard vm from Eclipse
  • Full debug support - break points/source code/step through etc
  • Install Composites into running newton from Eclipse
  • Log integration
